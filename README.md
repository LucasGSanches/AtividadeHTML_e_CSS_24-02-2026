# AtividadeHTML_e_CSS_24-02-2026
Atividade universitária da disciplina de desenvolvimento web sobre html e css

Objetivo da atividade: desenvolver um site hipotético de venda de "rj90".

**Descrição dos exercícios desenvolvidos:**

1. Estrutura básica em HTML: template básico com as tags DOCTYPE html, head, body, além da tag h1 e 4 tags p.

2. HTML semântico: o index.html foi composto por um header, uma nav com três li, um main, três section e um footer

3. CSS básico e seletores: a página teve sua cor mudada para uma cor branca(#f5f5f5), um destaque foi feito no cabecario e no rodape
por meio de classe, no h3 por tag e a imagem teve seu tamanho definido por id.

4. Box model: a tag nav foi modifica para border-box, junto com vários parâmetros de dimensionamento.
